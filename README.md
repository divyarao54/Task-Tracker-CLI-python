## Usage instructions

- Adding a new task:
  `python task_cli.py add "task"`
- Listing existing tasks
  `python task_cli.py list <status>`
- Updating existing task
  `python task_cli.py update <id> "updated task"`
- Changing status of existing task
  `python task_cli.py mark-<status> <id>`
- Deleting a task from the task list
  `python task_cli.py delete <id>`
